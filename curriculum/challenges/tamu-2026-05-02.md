# TAMU OpenClaw Workshop Challenge Prompts

## Challenge 1 - After Introduction

Post one thought from the morning session using **#TAMUopenclaw**.

Prompt idea:
> I’m at the OpenClaw workshop at Texas A&M learning how agents, memory, and orchestration work together. Today’s build: Beaver Badges.

## Challenge 2 - After Environment Setup

Create your first Markdown memory note in your OpenClaw workspace with:
- what you want to build
- what kind of agent would help you most
- one thing you want to learn today
- one kind of agent-created data worth saving

## Challenge 3 - After OpenClaw Setup

Ask OpenClaw:
> Based on my interests, which of the five workshop agents should I create first, and why?

Then save the answer into your notes.

## Challenge 4 - After Agent Setup

Define your own startup team in one sentence each:
- Market Researcher
- Idea Validator
- Coder
- Project Manager
- Outreach Assistant

Then ask your orchestrator what order those agents should be used in for a new startup idea, and turn the answer into a Markdown task list.

## Challenge 5 - During Beaver Badges Build

Ask your agents:
> What is the smallest MVP for Beaver Badges that still feels fun?

Then compare the answers and decide what belongs in the live build versus the roadmap.

After that, ask the team to improve the Beaver Badges README so someone new can understand the project goal, setup path, MVP, and next steps.

## Challenge 6 - Final Hour

Pick one next-step path:
- improve Beaver Badges for 30 days
- fork the idea into your own startup concept
- build your own command center and agent team

Then write a concrete commitment:
> In the next 30 days, I will ship ________ using my OpenClaw startup team.
