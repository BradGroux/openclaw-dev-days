# Startup Team Prompts

## Starter Prompts
- Given this startup idea, tell me which role should go first and write the prompt for them.
- We already finished research. What should the next role do before coding starts?
- Here are outputs from multiple roles. Synthesize them into the next best action.
- Keep this workshop team on track from idea to demo using the available agent roles.
- Turn this vague concept into a step-by-step agent workflow for the session.
