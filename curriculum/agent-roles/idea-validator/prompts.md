# Idea Validator Prompts

## Starter Prompts
- Pressure-test this startup idea and tell me if it is strong enough for a workshop MVP.
- Here is the feature list. Separate must-have MVP scope from later-stage ideas and things we should cut.
- Tell me the weakest assumptions in this concept and how we should de-risk them.
- Rewrite this idea into a tighter version with a clearer user and a clearer value proposition.
- If we only have 90 minutes, what is the sharpest version of this product we could realistically ship?
