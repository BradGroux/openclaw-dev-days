## Agent Team

### Process Mapper

**Role:** Turn the messy workflow into clear steps, decisions, inputs, outputs, and owners.

**Prompt starter:**

> Help me map this process into clear steps. Identify decision points, owners, inputs, outputs, handoffs, and bottlenecks. Ask clarifying questions before making assumptions.

### Security Reviewer

**Role:** Identify sensitive data, credential risks, operational risks, compliance concerns, and approval gates.

**Prompt starter:**

> Review this workflow for security, privacy, compliance, and operational risks. Create a guardrail checklist and identify what should not be shared with an AI tool.

### Documentation Writer

**Role:** Create the SOP, checklist, runbook, or briefing artifact.

**Prompt starter:**

> Turn this process into a practical SOP. Include prerequisites, steps, decision points, exceptions, rollback/escalation paths, and verification checks.

### Implementation Planner

**Role:** Turn the idea into tasks, milestones, verification gates, and a safe rollout plan.

**Prompt starter:**

> Create an implementation plan for improving this workflow. Start with low-risk steps, include verification gates, and keep humans in approval loops.

### Automation Scout (Optional)

**Role:** Find low-risk automation opportunities.

**Prompt starter:**

> Identify automation opportunities in this workflow. Rank them by impact, effort, and risk. Recommend the safest first automation experiment.
